Validaciones JS

Los campos se validarán de la siguiente manera

Ficha: se puede ingresar cualquier caracter

Nombre: solo se puede ingresar letras

Apellidos: solo se puede ingresar letras

Correo: solo se puede ingresar números

Teléfono: solo se puede generar

Género: solo se puede seleccionar un género

Ciudad: solo se puede seleccionar una ciudad

Habilidades: se tiene que seleccionar como mínimo 3 habilidades

Seleccione mínimo 3 habilidades
